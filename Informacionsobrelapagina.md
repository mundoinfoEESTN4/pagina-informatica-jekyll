Este proyecto fue desarrollado por los alumnos de 7mo 3ra del Departamento de Informática.  
El objetivo es aplicar los conocimientos de **HTML, CSS y JavaScript** para crear sitios web funcionales y bien estructurados.

Incluye un sistema de noticias dinámico, proyectos destacados y un feed RSS actualizado automáticamente.
