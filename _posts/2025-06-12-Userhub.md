# ¿Que es UserHub?
UserHub es una plataforma web diseñada para gestionar usuarios y proyectos de forma sencilla, moderna y conectada. Su propósito es servir como un nexo central entre personas y sus trabajos, ofreciendo una interfaz accesible y visualmente atractiva.

El sistema adopta una arquitectura cliente-servidor. El backend está desarrollado en Django (Python) con base de datos relacional (SQLite o MySQL), mientras que el frontend utiliza HTML, CSS y JavaScript con un diseño moderno. Además, incorpora servicios de logging mediante sockets TCP y se encuentra desplegado en la nube a través de Render.

Gracias a su estructura modular, su enfoque escalable y el uso de tecnologías actuales, UserHub se presenta como una herramienta ideal para la administración digital de proyectos, especialmente en entornos educativos y productivos, impulsando prácticas profesionalizantes y experiencias reales de trabajo.
